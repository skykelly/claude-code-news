## 활용 분야별 분류

### 1. 풀스택 웹 개발
- React/Next.js 앱, 대시보드, SaaS 서비스
- JS/TS 프론트엔드 + Python 백엔드 조합이 가장 많음

### 2. 레거시 시스템 현대화
- 금융기관 메인프레임 앱 현대화
- 규제 컴플라이언스 분석 및 보안 개선

### 3. 데이터 & AI/ML
- 연구용 실험 코드 → 프로덕션 시스템 전환
- CSV 데이터 자동 분석 및 리포트 생성

### 4. DevOps & 인프라 자동화
- CI/CD 파이프라인 구축
- 서버 설정, 구성 자동화

### 5. 비즈니스 프로세스 자동화
- 수백 개 광고 CSV 처리 → 성과 분석 → 신규 변형 자동 생성
- 고객 통화 내용 자동 종합
